Team Gold - Handbuch

System-Voraussetzungen

Damit das Spiel auf Ihrem Computer laufen kann, muss
  1. Java Version 8 installiert sein
  2. Der java-Befehl in der Umgebungsvariablen PATH eingetragen sein
  3. OpenGL Version 3.1 unterstütz werden (wird von fast jeder Grafikkarte erfüllt)

Installation

Um das Spiel zu installieren entpackten Sie die Game.zip in den gewünschten Installations-Ordner.

Spiel starten

Um das Spiel zu starten gibt es zwei Möglichkeiten
  1. Per Konsole in den Game-Order navigieren:
    cd Pfad/Zum/Game
    Anschließend den Java-Interpreter aufrufen:
    java -jar -Djava.library.path=./lib Game.jar
  2. Unter Windows kann man auch die starter.bat doppelklicken
